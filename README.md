# 🎬 Movie Recommender System

🚀 An end-to-end Machine Learning project that recommends movies based on user preferences using a **content-based filtering approach**. This system analyzes movie similarities and provides personalized recommendations along with posters using an external API.

---

## 📌 Project Overview

The Movie Recommender System is designed to help users discover movies similar to their interests. It leverages **machine learning techniques**, **data preprocessing**, and **similarity computation** to generate accurate recommendations.

The application also includes a **user-friendly interface built with Streamlit**, making it interactive and easy to use.

---

## 🔥 Features

- 🎯 Recommend top 5 similar movies
- 🧠 Content-based filtering approach
- 🖼 Fetch movie posters using TMDB API
- 📊 Data preprocessing and feature engineering
- ⚡ Fast and interactive UI using Streamlit

---

## 🛠 Tech Stack

- **Programming Language:** Python  
- **Libraries:** Pandas, NumPy, Scikit-learn  
- **Visualization:** Matplotlib  
- **Framework:** Streamlit  
- **IDE:** PyCharm  
- **API:** TMDB (The Movie Database API)  

---

## ⚙️ How It Works

1. **Data Collection**
   - Movie dataset containing titles, genres, keywords, cast, and crew

2. **Data Preprocessing**
   - Handling missing values  
   - Text cleaning and transformation  

3. **Feature Engineering**
   - Combining relevant features into a single text column  
   - Applying vectorization techniques (Bag of Words)

4. **Similarity Calculation**
   - Cosine similarity used to measure similarity between movies  

5. **Recommendation System**
   - Fetches top 5 most similar movies based on input  

6. **UI Integration**
   - Built using Streamlit for user interaction  
   - Displays movie names and posters
